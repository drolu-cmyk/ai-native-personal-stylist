# Code of Conduct

This project should stay practical, respectful, and safe for contributors and users.

## Expected behavior

- Be respectful.
- Keep feedback specific and useful.
- Protect private user data.
- Do not post credentials, private images, voice samples, billing data, or live location trails.
- Keep discussions focused on the product and code.

## Unacceptable behavior

- Harassment or personal attacks.
- Posting private data.
- Sharing exploit details in public issues.
- Spam or off-topic promotion.

Report security issues through the security policy.

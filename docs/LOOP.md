# Loop

The product loop is simple.

1. The user gives the plan.
2. The app checks saved clothes.
3. The app checks weather and place context when allowed.
4. The app returns one outfit.
5. The user keeps it or asks for another mix.
6. The next result uses that response.

## Olu AI Skills
